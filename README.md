O projeto é um programa em C# que simula o controle de um estoque de produtos, utilizando uma lista encadeada dinâmica como estrutura de dados.

O sistema permite cadastrar produtos informando:
  -Id do produto
  -Nome do produto
  -Quantidade em estoque

Os produtos são armazenados em uma lista, onde é possível gerenciar o estoque por meio de um menu interativo.

O programa possui as seguintes funções:
  -Adicionar um novo produto ao estoque
  -Remover um produto pelo seu Id
  -Listar todos os produtos cadastrados
  -Pesquisar um produto específico pelo Id

O projeto é dividido em classes que organizam o funcionamento do sistema:
  Produto – armazena os dados do produto e possui funções para exibir informações e editar dados 
  Celula – representa cada nó da lista encadeada
  Lista – controla a inserção, remoção, listagem e busca dos produtos
  Program – gerencia o menu e a execução do sistema

O objetivo principal é praticar estrutura de dados (lista encadeada), programação orientada a objetos e interação via console.
